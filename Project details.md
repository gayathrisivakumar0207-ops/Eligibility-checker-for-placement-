PROJECT TITLE

Placement Eligibility Checker

PROJECT OVERVIEW

The Placement Eligibility Checker is a web-based application that helps students determine whether they are eligible for campus placement drives based on company requirements such as CGPA, active arrears, department, and graduation year. The system automatically verifies student details and displays the eligibility status instantly.

PROBLEM STATEMENT

Many students are uncertain about their eligibility for campus placements. Placement officers often spend a lot of time manually checking student records against company criteria. This process is time-consuming and may lead to errors. Therefore, an automated Placement Eligibility Checker is required to provide quick and accurate eligibility verification.

PROJECT OBJECTIVE

Main Objective:
To develop a system that automatically checks whether a student is eligible for placement opportunities based on predefined company requirements.

Specific Objectives:
1. Collect student academic details.
2. Validate CGPA and arrear information.
3. Compare student data with company eligibility criteria.
4. Display eligibility status instantly.
5. Reduce manual verification work.
6. Improve accuracy and efficiency.

FEATURES

1. Student Registration
2. Eligibility Checking
3. CGPA Validation
4. Arrear Verification
5. Instant Result Display
6. User-Friendly Interface

INPUT PARAMETERS

1. Student Name
2. Register Number
3. Department
4. CGPA
5. Active Arrears
6. Graduation Year

OUTPUT

Eligible – If all criteria are satisfied.

Not Eligible – If any criteria are not satisfied.
